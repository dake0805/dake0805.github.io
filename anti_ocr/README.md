# Anti-OCR

https://www.anti-ocr.ml/

這是一個用於對抗文字識別的工具

本工具可以將文字通過隨機畫點/線以及旋轉，轉換爲可以幹擾OCR識別的圖片

2018.1.28更新:

增加用戶修改畫點密度/大小，線條粗細的功能
